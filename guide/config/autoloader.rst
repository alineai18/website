Autoloader
==========

Aplus Framework App Project Autoloader Config.

This is an example of the *config/autoloader.php* configuration file delivered
with the App project:

.. code-block:: php

    return [
        'default' => [
            'namespaces' => [
                'App' => APP_DIR,
            ],
            'classes' => [],
        ],
    ];
