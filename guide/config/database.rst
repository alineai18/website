Database
========

Aplus Framework App Project Database Config.

This is an example of the *config/database.php* configuration file delivered
with the App project:

.. code-block:: php

    return [
        'default' => [
            'username' => 'root',
            'password' => 'password',
            'schema' => 'framework-tests',
            'host' => 'localhost',
            'logger_instance' => 'default',
        ],
    ];
